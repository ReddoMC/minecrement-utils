# minecrement-utils
Adds many QoL features to make your life easier, and to help AFK! You can be tabbed out and the mod will still do it's thing!

Feature list:
/calculatemine <mspeed> <pick_power> (calculate best mine for coins)
/smartwarp <mine/cave> to quick warp to places
Auto-Reconnect
Auto GG
Auto-Ability (Auto detects your abilities!)
Webhook sending various things, very configurable
Reset Helper to make rebirthing and prestiging easier.
